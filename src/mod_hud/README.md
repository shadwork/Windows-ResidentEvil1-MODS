# Resident Evil 1 HUD Overlay

Shows a compact real-time HUD during gameplay:
- inventory item icons
- ammo quantities for weapons
- currently equipped weapon highlight
- equipped weapon and ammo display
- player health ECG

The HUD can be customized in `mod_hud.ini`. If the file is missing, the mod
creates a default config next to `mod-hud.dll` on first run.

This mod is intended for the Classic REbirth / RE1 mod loader setup.
